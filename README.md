# CreditClaw Plugin Marketplace

Claude Code / Claude Cowork plugin marketplace for CreditClaw.

## Install

```bash
# Add the marketplace
/plugin marketplace add CreditClaw/marketplace

# Install CreditClaw
/plugin install creditclaw@creditclaw
```

## Available Plugins

| Plugin | Description |
|--------|-------------|
| `creditclaw` | Payment card with spending controls for AI agents |
